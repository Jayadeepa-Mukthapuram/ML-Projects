Machine Learning using Logistic Regression

## Overview
This project implements a machine learning model to classify Iris flower species based on flower measurements.  
Logistic Regression is used for classification and the model achieves an accuracy of 96%.

## Dataset
- Iris Dataset (Scikit-learn built-in)
- Total samples: 150
- Classes:
  - Iris Setosa
  - Iris Versicolor
  - Iris Virginica
- Features:
  - Sepal length
  - Sepal width
  - Petal length
  - Petal width

## Algorithm Used
- Logistic Regression
- Suitable for multi-class classification
- Simple and effective for small datasets

## Model Performance
- Accuracy: 0.96 (96%)

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Result
The Logistic Regression model successfully classifies Iris flower species with high accuracy.
